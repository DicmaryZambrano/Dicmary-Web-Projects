Overview

I will explore creating an application that allows users to explore video games across different platforms, save their favorite games to a personal collection, and receive game recommendations based on their preferences.

Site Name

DZ Games

Purpose

This website is geared towards people that like games or want to know more about games, By utilizing the IGDB API it displays a list of up to 100 games with their categories, videos, screenshots, description, storyline, etc.

Audience

The audience for this website is individuals that want to discover new games to add to their collection people that want to know more about a certain game like the rating, the platforms the game is in, when it was published, if there are any new extensions or new trailers, etc.

Scenarios

  • What are the popular games right now?
  • What are the latest game events?
  • What new adventure, horror, or simulator games are there?
  • What high rated pc, PS4 or Xbox games are there?
  • Where can I save the games I liked?
  • What is the general information of the game?
  • Where could I see more information about a game?

Core Features

  • Users will be able to search for games and view detailed information
  • Save favorite games to local storage
  • Receive game recommendations
  • Filters for genres, platforms, and other criteria will be included to help users find games they love

Data sources

Local Storage (for wishlist) API: https://api-docs.igdb.com/#getting-started

Dynamics

The header will be displayed dynamically; all game information is displayed dynamically with the help of the Api

Animation ideas

Animated hamburger menus, animated gallery, hover effects, animated search bar, animation when adding a new favorite, sliding notifications

Classes

  • Game Details: gets the game data using the game id and renders it to the page, information like platforms, ratings, covers, screenshots, videos, and related dlc are displayed. Using the game keywords a section with similar games is generated.
  • Wishlist: Gather information from local storage to render a list of favorite games
  • Favorites: uses local storage to gather the selected favorite breeds, creates the grid template, renders the display
  • Event List: Populates and renders a list of game events with the template, target element, and list of objects provided.
  • SearchEngine: Uses user input to search for games in the IGDB API that could be related to the keywords
  • Game List: Populates and renders a list of games with the template, target element, and list of objects provided.
  • ApiServices: Takes care of all the Api post and get requests for other modules.
  • Utilities: Takes care of utilities like populating headers, creating notifications, rendering, etc.

Modules

  • GameDetails
  • GameList
  • Utilities
  • ApiServices
  • SearchEngine
  • Wishlist
